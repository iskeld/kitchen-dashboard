
##Requirements
* [Ruby 1.9+](https://www.ruby-lang.org/en/downloads/) 
* Bundler 
```
$ gem install bundler
```
##To build
```
$ bundle
```
##To run on localhost
[localhost:3030/sampletv](http://localhost:3030/sampletv)
```
$ dashing start
```
